To run the example:

1. Start Python and change current working directory to the directory that contains the 'runOptionpricing.py' file

2. In Python 2, run:

execfile("runOptionpricing.py")
runDemo()

OR 

In Python 3, run:

from runOptionpricing.py import *
runDemo()
