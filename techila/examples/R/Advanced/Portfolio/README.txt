1. Create an environment variable TECHILA_SDKROOT and set it to point to your Techila SDK 'techila' directory location. 

2. Start GNU R and change current working directory to the directory that contains the 'Techila-PortfolioGeneration-ModelCode.R' file

3 In GNU R, run:

source("Techila-PortfolioGeneration-ModelCode.R")
