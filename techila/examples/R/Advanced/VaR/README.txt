More information about this example can be found here:

http://www.techilatechnologies.com/var

To run:

1. Create an environment variable TECHILA_SDKROOT and set it to point to your Techila SDK 'techila' directory location. 

2. Start GNU R and change current working directory to the directory that contains the 'run_VaR_cloudfor.R' file

3 In GNU R, run:

source("run_VaR_cloudfor.r")

res <- run_VaR_cloudfor()
